- venv, conda란?
- 목적
  - 파이썬 가상환경을 관리하기위한 도구
- venv
  - 파이썬 3.3버전부터 제공되는 가상환경 도구
  - 파이썬 프로젝트 간 독립된 가상 환경 구성 가능
  - 각 프로젝트 별 패키지를 별도의 가상환경에 설치하며 프로젝트 간 충돌 방지
  - 표준 라이브러리에 포함되어 있으며, 파이썬 인터프리터도 제공
- conda
  - 파이썬 패키지 관리도구인 Anaconda의 일부로 개발된 가상 환경 도구
  - 파이썬 패키지 뿐만아니라 다양한 프로그래밍 언어 패키지 관리 가능
